¿Cómo funciona?

Hemos desarrollado un sistema sencillo, fiable y (sobre todo) bonito. Solo tendrás que seguir los siguientes pasos:

  • Paso 1

    Configuración básica

    En este paso nos daréis la información básica para montar una primera versión de vuestra web.

    Que haremos:
    • Crear vuestro dominio personalizado
    • Revisar la web de demo e identificar las secciones que os gustaría incluir
    • Identificar el orden de las secciones
    Más información
  • Paso 2

    Configuración detallada

    En este paso haremos todas las iteraciones que sean necesarias para que la web quede configurada a vuestro gusto

    Para cada sección necesitaremos que nos proporcionéis:
    • Cambios en los textos
    • Cambios en las imágenes
    • Otros cambios o sugerencias
    Más información
  • Paso 3

    Seguimiento

    Ahora ya solo os tenéis que preocupar de hacer seguimiento de los invitados que se han apuntado o de los regalos que os hayan hecho

    ¿Como lo haremos?:
    • Os daremos acceso a un excel de seguimiento de invitados
    • Podréis cambiar sobre la marcha información relevante para vuestros invitados como:
      • Hoteles
      • Lista de regalos
      • Peluquerías
      • Otros lugares de interés
    Más información
